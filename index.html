🚀 Node.js Events and Timers

🎯 Events & Event Listeners

Hover → Enlarge

hoverBox.addEventListener('mouseenter',()=>{ hoverBox.classList.add('enlarge'); }); hoverBox.addEventListener('mouseleave',()=>{ hoverBox.classList.remove('enlarge'); });

Click → Rotate

clickBox.addEventListener('click',()=>{ clickBox.classList.toggle('rotate'); });

Double Click → Color Change

dblBox.addEventListener('dblclick',()=>{ dblBox.style.background = dblBox.style.background==='red' ? '#38bdf8':'red'; });

Click Box A → Activates B (and vice versa)

boxA.addEventListener('click',()=>{ boxB.classList.toggle('active'); }); boxB.addEventListener('click',()=>{ boxA.classList.toggle('active'); });

Press any key → Skew

document.addEventListener('keydown',()=>{ keyBox.classList.add('skew'); setTimeout(()=>{ keyBox.classList.remove('skew'); },300); });

⏱ Timers

Hover → stays enlarged 5s after leaving

timeoutBox.addEventListener('mouseenter',()=>{ clearTimeout(timer); timeoutBox.classList.add('enlarge'); }); timeoutBox.addEventListener('mouseleave',()=>{ timer=setTimeout(()=>{ timeoutBox.classList.remove('enlarge'); },5000); });

Hover → heartbeat animation

intervalBox.addEventListener('mouseenter',()=>{ interval=setInterval(()=>{ intervalBox.classList.toggle('enlarge'); },500); }); intervalBox.addEventListener('mouseleave',()=>{ clearInterval(interval); intervalBox.classList.remove('enlarge'); });
